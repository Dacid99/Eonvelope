Contact
=======

For issues, feature requests, merge requests and similar topics please get in touch
via the issue trackers on `gitlab <https://gitlab.com/Dacid99/Eonvelope/-/issues>`_ and `github <https://github.com/Dacid99/Eonvelope/issues>`_.
Requests for adding a missing language should also be submitted via an issue.
If you have found a security-related problem, please use the email instead.

Other subjects, like media inquiries, can be discussed via email.
The official email address is `eonvelope@mailo.com <mailto://eonvelope@mailo.com>`_.
